# Сервис для работы с табличными отчетами (Excel) с поддержкой обновлений и статистики качества 

## Запуск 

- В корневой директории создать файл .env и заполнить переменные из .env.template
- Запустить командой 
```
docker-compose up --build
```

## Запуск тестов

- В корневой директории создать файл .env и заполнить переменные из .env.template

- Запустить командой 
```
docker-compose up --build -в
```
- Запустить тесты make tests
